# Sales Performance Dashboard

## Overview
An interactive Tableau dashboard analyzing sales trends, 
regional performance, and product profitability using the 
Sample Superstore retail dataset.

## Features
- Sales and Profit KPI cards
- Monthly Sales Trend analysis
- Sales by Region comparison
- Top 10 Products by Sales
- Profit vs Sales scatter plot by Sub-Category
- Calculated fields: Profit Ratio
- Interactive filters for Product Name and Sub-Category

## Tools Used
- Tableau Public

## Dataset
Sample Superstore Dataset (US Retail Sales Data 2014-2017)
